Y U NO BIG?

Calorie counting should be quick and get out of your way.
the current websites/apps sucks since it takes too much time and not always intuitive to use.
this site is my attempt in makeing things simpler.

The Website

install

install node.js from http://nodejs.org/
npm install # install dependencies

setup

make db-seed  # load food to DB

run

node server.js

develop

I use browserify to enjoy CommonJS awesomeness in the client side js. --watch is not working so i run it manually whenever i change client side js files: ./node_modules/browserify/bin/cmd.js public/js/controllers.js public/js/calc.js -o public/js/bundle.js

I use stylus for css

get big

http://localhost:3000

tech stack

  • Node.js - server side code
  • MongoDB - DB
  • AngularJS - client side MVC framework
  • Zombie.js - end-2-end testing
  • browserify - CommonJS for client-side js
  • stylus for nicer css
